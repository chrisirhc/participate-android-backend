Edit Class:


Due to a bug in the parsing for the times, if your class starts/ends in the hour of 12PM (e.g. 12.30PM), please leave out the "PM" so it looks like "12:00" or "12:30". If the class starts at midnight, use "0:00" or "0:12" and so forth. Otherwise, always specify PM / AM, there is no 24H parsing.
